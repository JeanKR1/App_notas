# 📝 Smart Notes App - Full Stack con Inteligencia Artificial

¡Bienvenido a **Smart Notes App**! Esta es una aplicación de gestión de notas diseñada para estudiantes, potenciada por Inteligencia Artificial (Google Gemini) para mejorar la productividad académica.

## 🚀 Características Principales

- **Gestión de Notas:** Creación, edición y visualización de notas en tiempo real.
- **Integración con IA (Gemini 3 Flash):**
  - ✨ **Resumen Automático:** Genera síntesis profesionales de contenidos extensos.
  - ✨ **Corrección Ortográfica:** Optimiza la gramática y fluidez del texto.
  - ✨ **Sugerencia de Etiquetas:** Analiza el contenido para categorizar las notas automáticamente (ideal para temas como Redes, Cálculo o TI).
- **Arquitectura Full Stack:** Separación clara entre el cliente (Frontend) y el servidor (Backend).

## 🛠️ Tecnologías Utilizadas

### Frontend
- **React.js:** Interfaz de usuario dinámica y responsiva.
- **CSS3:** Estilos personalizados para una experiencia de usuario limpia.
- **JavaScript (ES6+):** Lógica de la aplicación y consumo de APIs.

### Backend
- **Node.js & Express:** Servidor robusto para el manejo de peticiones.
- **Google GenAI SDK:** Integración con los modelos más recientes de Gemini (v3 Flash).
- **Dotenv:** Gestión segura de variables de entorno (API Keys).
- **CORS:** Configuración de seguridad para comunicación entre puertos.
